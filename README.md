This program features Encryption and Decryption as well as digital signing using the RSA protocol nad implemented using C++.
It works for 1 word messages only (since the point is to implement the basics of the algorithm)."
Upon executing the program the user can pick 1 entering the Encryption. Enter p, q the word to be encrypted and the e, the word will be encoded and encrypted.
Make a note of the d the n and the cipher text afterwards to put them as parameters in the Decryption function which will turn the cipher text into the original encoded message.
